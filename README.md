# RihlaBot 🌍

RihlaBot est un assistant de voyage marocain intelligent développé avec Botpress.

## Description
Bot conversationnel qui aide les voyageurs à découvrir le Maroc.

## Technologies utilisées
- Botpress (plateforme de chatbot)
- GitHub (intégration webhook)

## Accès au bot
- Plateforme : Botpress Cloud
- Statut : Online ✅
